# 📊 Flipkart Sales Dashboard

## 🚀 Project Overview

Designed and developed a professional business intelligence dashboard in Microsoft Excel to analyze Flipkart sales performance. The dashboard transforms raw sales data into actionable insights using KPI cards, Pivot Tables, Pivot Charts, and interactive visualizations.

## 🎯 Business Problem

E-commerce companies generate large volumes of sales data every day. This project helps business stakeholders understand:

* Which product categories generate the highest revenue
* Which regions contribute most to sales
* Monthly sales growth patterns
* Profitability across product categories
* Overall business performance through KPIs
